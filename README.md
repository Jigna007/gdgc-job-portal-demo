# Recrewit Job Portal

Simple job portal built for both job seekers and recruiters. Switched between modes to see different features.

## What it does

**Job Seekers:**

- Browse available jobs with filtering and search
- View detailed job descriptions
- Apply to positions
- Track application status

**Recruiters:**

- Post new job listings
- Manage job status (active, paused, closed, etc.)
- View applications and applicant profiles
- Track job performance metrics

## Files

- `index.html` - Main page structure
- `styles.css` - All the styling
- `script.js` - App functionality and data handling

## Running it

Just open `index.html` in any browser. No server needed.

## Features

- Role switching between job seeker and recruiter
- Job filtering by type, salary, experience level, company size
- Application management system
- Candidate profile viewing
- Responsive design that works on mobile

The job data comes from a fake API for demo purposes. In production you'd connect to a real database.
